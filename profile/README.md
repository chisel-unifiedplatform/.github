## What we build

Orphix is a unified agentic-AI platform. Instead of stitching together a dozen point
tools, teams use Orphix to design an agent once and ship it across every channel —
a phone call, a chat widget, or a headless workflow — backed by the same knowledge,
tools, and guardrails.

| Capability | What it does |
| --- | --- |
| 🎙️ **Voice Agents** | Low-latency, real-time voice agents for inbound and outbound calls, built on a streaming call server. |
| 💬 **Chat Agents** | Embeddable chat widget and messaging agents that share context with your voice agents. |
| 🔌 **MCP Integrations** | 280+ Model Context Protocol servers, giving agents secure, typed access to the tools and data they need. |
| ⚙️ **Workflow Automation** | Visual, event-driven workflows to connect agents to your systems and automate multi-step processes. |
| 🧠 **Knowledge Base** | Retrieval-backed knowledge so agents answer from your content, not guesses. |
| 🧑‍💼 **Human-in-the-loop** | Agent panels and live handoff so a human can step in whenever it matters. |
| 🏢 **Multi-tenant** | Isolated tenants, connections, and data — built for teams running many agents. |
