# .github

This is the **organization-level `.github` repository** for
[Orphix](https://github.com/chisel-unifiedplatform). It holds two kinds of shared content:

### 1. Organization profile
- [`profile/README.md`](profile/README.md) — the public page shown at
  [github.com/chisel-unifiedplatform](https://github.com/chisel-unifiedplatform).

### 2. Default community health files
These are inherited by **every repository** in the org that doesn't define its own:

| File | Purpose |
| --- | --- |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | How to propose changes and open PRs |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Community standards |
| [`SECURITY.md`](SECURITY.md) | How to report vulnerabilities |
| [`SUPPORT.md`](SUPPORT.md) | Where to get help |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) | Bug report & feature request forms |
| [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md) | Default PR checklist |

> A file placed in an individual repo always overrides the default here.

_See GitHub's docs on [default community health files](https://docs.github.com/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file)._
