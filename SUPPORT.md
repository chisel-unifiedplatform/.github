# Support

Need help with Orphix? Here's where to go.

## Product & account help

- **Website:** [orphix.ai](https://orphix.ai)
- **General enquiries:** [hello@orphix.ai](mailto:hello@orphix.ai)

## Questions about the code

- Check the **README** of the specific repository first.
- Search existing **issues** — your question may already be answered.
- If not, open a new issue using the **Bug report** or **Feature request** template.

## What's *not* a support channel

- **Security vulnerabilities** → follow the [Security Policy](SECURITY.md), not a public issue.

We do our best to respond, but please note that response times for community issues are
best-effort and not guaranteed.
